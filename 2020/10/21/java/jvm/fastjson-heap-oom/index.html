1. 问题描述

temp.toJSONString()报错

因为代码是里面用了递归，一开始以为是代码问题，debug了许久，没发现哪里逻辑错了。
想着之前学过JVM，就试着通过堆快照分析哪里错了

2. 堆快照分析
