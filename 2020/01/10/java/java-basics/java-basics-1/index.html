#ArrayList和LinkedList

数据的更新和查找

ArrayList的所有数据是在同一个地址上,而LinkedList的每个数据都拥有自己的地址.所以在对数据进行查找的时候，由于LinkedList的每个数据地址不一样，get数据的时候ArrayList的速度会优于LinkedList，而更新数据的时候，虽然都是通过循环循环到指定节点修改数据，Arraylist快，LinkedList慢，需要改指针。

数据的增加和删除

数组增加，后面的都需要+1 ，删除-1
链表只需要改节点
单一元素的增删，尤其是在List末端增删一个元素，二者效率不相上下

时间复杂度

ArrayList 是线性表（数组）

  • get() 复杂度 O(1)
  • add(E) 添加元素，直接在后面添加，复杂度O（1）
  • add(index, E) 添加元素，在第几个元素后面插入，后面的元素需要向后移动，复杂度O（n）
  • remove（）删除元素，后面的元素需要逐个移动，复杂度O（n）

LinkedList 是链表的操作

  • get() 获取第几个元素，依次遍历，复杂度O(n)
  • add(E) 添加到末尾，复杂度O(1)
  • add(index, E) 添加第几个元素后，需要先查找到第几个元素，直接指针指向操作，复杂度O(n)
  • remove（）删除元素，直接指针指向操作，复杂度O(1)
